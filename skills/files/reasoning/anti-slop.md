# Anti-Slop Writing Standards

Academic papers fail not from being wrong, but from being vague.
Apply these rules to every sentence you write.

## Banned phrases (replace immediately)

| Slop | Replace with |
|------|-------------|
| "plays a crucial role" | state what the role IS |
| "it is worth noting" | just note it |
| "interestingly" | say WHY it is interesting |
| "significant implications" | list the implications |
| "this paper contributes to" | state the contribution precisely |
| "future research could" | only if you have a specific suggestion |
| "this is an important question" | the reader decides that |

## Structure each paragraph to carry one claim

1. Topic sentence: the claim
2. Evidence: data, equation, citation
3. Interpretation: what this means for the argument

## Numbers are not self-explanatory

Every coefficient, percentage, and test statistic needs:
- Units
- Baseline for comparison  
- Economic interpretation

Bad: "The coefficient is 0.023."
Good: "A one-standard-deviation increase in X raises Y by 2.3 percentage points (13% of the sample mean)."

## Precision over hedging

Do not write "may potentially suggest" when you mean "suggests."
Hedge only when genuinely uncertain, and state the source of uncertainty explicitly.
